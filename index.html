Whiley

Whiley is a programming language particularly suited to safety-critical systems. It is a hybrid object-oriented and functional programming language which employs extended static checking to eliminate errors at compile time, including divide-by-zero, array out-of-bounds and null dereference errors.

Whiley compiles to the Java Virtual Machine and is fully inter-operable with existing Java applications. Whiley will operate on any system that can run Java including Microsoft Windows, Mac OS X and Linux. Whiley is released under the New BSD License.

To find out more, visit http://whiley.org.

Installation

There are several ways to get started with Whiley:

  1. Install the Eclipse Plugin.

  2. Download and run the Executable Jar.

  3. Download and setup the Whiley Development Kit.

The Eclipse plugin is the easiest way to get started with Whiley (see instructions below). If you don't use Eclipse, another simple option is to download and run the executable Jar. Finally, developers may like to download the Whiley Development Kit (WDK), which includes the source code for the compiler/runtime and various scripts to simplify compiling and running Whiley programs from the command-line.

Whiley Eclipse Plugin (Wyclipse)

To install the Whiley Eclipse plugin start Eclipse and select "Install New Software" (normally found under "Help"). Then, add http://whiley.org/eclipse as a "software site", after which you should find that "Whiley Eclipse Plugin" becomes an option to install. At this point, "select all" and follow the instructions to install the plugin.

For more info on Wyclipse, see http://whiley.org/tools/wyclipse/.

Whiley Executable Jar

First, download the most recent stable version of the executable jar (wyjc-all-vX.Y.Z.jar where vX.Y.Z is the version number) from http://whiley.org/downloads. Then you can compile Whiley files as follows:

% cd examples
% java -jar wyjc-all-vX.Y.Z.jar -bp wyjc-all-vX.Y.Z.jar hello-world.whiley

Once the Whiley program has successfully been compiled, you can run it as follows:

% java -cp "wyjc-all-vX.Y.Z.jar;." hello-world
Hello World

Note: under Mac OS and UNIX you will want to replace the ; with a :.

Whiley Development Kit

To install the Whiley Development Kit, first download the latest stable version from http://whiley.org/downloads. Then, unpack the tarball into an appropriate directory (as you've probably already done). You then need to make sure that PATH environment variable points to the bin directory, where several scripts are provided for running Whiley. (Don't worry if you don't know what the PATH is, as you can instead run the jar directly - see below).

With the PATH setup correctly, you should be able to run the wyjc command to compile Whiley programs as follows:

% cd examples
% wyjc hello-world.whiley

At this point, you can run the program using the wyj command:

% wyj hello-world
Hello World

Note: the wyj command is just a wrapper for java -cp lib/wyrt-vX.Y.Z.jar.

If you're running on a Windows system without cygwin, or you're having trouble getting the wyjc script to work, then a useful alternative is to run the jar file directly. You can do this as follows:

% cd examples
% java -jar ../lib/wyjc-vX.Y.Z.jar -bp ../lib/wyrt-vX.Y.Z.jar hello-world.whiley

Then, to run the program you would do this:

% java -cp "../lib/wyrt-vX.Y.Z.jar;." hello-world
Hello World

Note: under Mac OS and UNIX you will want to replace the ; with a :.
