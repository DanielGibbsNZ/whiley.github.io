Whiley Eclipse Plugin (Wyclipse)

Getting Started

Get started with Wyclipse by downloading the latest version of the project from Github by either:

  • Forking the repository on Github.
  • Downloading a tarball or zipball of the repository.
  • Cloning the repository: git clone https://github.com/Whiley/Wyclipse.git

Wyclipse can be imported into Eclipse by using the following steps:

  • Click File -> New -> Java Project.
  • Enter the project name as "Wyclipse".
  • Uncheck Use default location.
  • Set the location to the location of the Wyclipse project.
  • DO SOMETHING HERE TO SETUP THE MODULES.
  • Click Finish.
